# Bradford Household Composition Trends

This repository contains code for analysing changes in household composition across the 2001, 2011 and 2021 Censuses at ward level. The workflow harmonises household composition categories across census years, maps LSOA-level data to 2024 ward boundaries, and calculates ward-level counts and percentages for comparison over time.

The analysis includes:

- harmonisation of household composition categories across Census classifications
- allocation of LSOA-level values to 2024 wards
- calculation of ward-level household composition counts and percentages
- trend tables and visualisations showing changes between Census years
